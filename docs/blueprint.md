# **App Name**: Circuito Control

## Core Features:

- Pilot Registration Form: An optimized and functional web-based form for pilots to register their details for the event.
- Google Sheets Database Integration: Store all registered pilot data in a structured Google Sheet for easy access and management, allowing real-time updates.
- Automated Email Confirmation: Automatically send confirmation emails to pilots upon successful registration, including essential event details.
- WhatsApp Notification Integration: System to send automated or manual notifications to pilots via WhatsApp, covering updates or confirmations.
- Unique QR Code Generation: Generate a unique QR code for each registered pilot, containing their identifier for quick on-site validation.
- On-site QR Scanning & Validation Tool: An intuitive interface for event staff to scan pilot QR codes and validate their entry in real-time.
- Communication Drafting Assistant: A generative AI tool that assists in drafting personalized email and WhatsApp messages for pilots based on event updates or common inquiries.

## Style Guidelines:

- Primary color: A vibrant yet professional blue (#3366CC) to convey technology and precision in a dark theme.
- Background color: A very dark, almost black, but slightly bluish tone (#181B1F) to create a sophisticated and focused dark user interface.
- Accent color: A bright and energetic cyan (#64D2F0) for call-to-actions and to highlight crucial information, creating strong contrast with the primary and background colors.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable appearance, ensuring clarity across all app sections.
- Utilize a consistent set of clean, minimalist line-art icons that clearly represent actions such as 'register,' 'scan,' 'confirm,' and 'send message'.
- Employ a spacious and organized layout with clear data hierarchy, ensuring form fields are easy to navigate and scanned information is immediately prominent.
- Incorporate subtle, quick animations for form submissions, successful QR scans, and navigation transitions to provide instant user feedback and a sense of responsiveness.
